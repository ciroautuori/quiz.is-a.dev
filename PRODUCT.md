# Product

## Register

product

## Users
Beginners learning Python programming, computer science students, and self-taught developers looking for an engaging, gamified way to practice Python logic and syntax.

## Product Purpose
Python Quest: Think Python Lab is an interactive educational web platform featuring quizzes, code inspection, output prediction, AI tutoring, skill tree progression, and real-time Python sandboxing.

## Brand Personality
Educational, crisp, empowering, modern developer laboratory feel with vibrant Catppuccin accents and dark-mode elegance.

## Anti-references
Generic SaaS light mode templates with gray body text, noisy glassmorphism defaults, cramped text contrast, and unstyled forms.

## Design Principles
1. **Readable Code First**: Syntax highlighting and terminal views must be crisp, high-contrast, and keyboard navigable.
2. **Clear Gamification Feedback**: Streaks, XP badges, and difficulty levels use distinctive, accessible Catppuccin color tokens.
3. **Responsive Tactile Layout**: Every interaction (modals, navs, quiz choices) feels instantaneous, accessible, and balanced on all screen sizes.
4. **Accessible Contrast & Motion**: Meets WCAG AA contrast standards with smooth, prefers-reduced-motion safe animations.

## Accessibility & Inclusion
Targeting WCAG AA compliance. Full keyboard accessibility for code views, interactive quizzes, and modal dialogs. High contrast body text against Catppuccin backgrounds.
